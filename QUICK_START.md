## 📝 操作指南

### 添加新项目

1. 点击"项目管理"标签
2. 点击"+ 添加项目"按钮
3. 填写项目信息：
   - 项目标题
   - 项目描述
   - 负责角色
   - 项目周期
   - 是否私密
4. 点击"保存"

### 编辑项目

1. 在项目列表中找到要编辑的项目
2. 点击"编辑"按钮
3. 修改项目信息
4. 点击"保存"

### 删除项目

1. 在项目列表中找到要删除的项目
2. 点击"删除"按钮
3. 确认删除

### 查看统计

1. 点击"概览"标签
2. 查看各项数据统计：
   - 项目数量
   - 技能数量
   - 媒体数量
   - 照片数量

## 🔄 如何重启服务

### 停止服务

按 `Ctrl + C` 停止当前运行的服务

### 重新启动

**Windows:**
```bash
双击 start.bat
```

**Linux/Mac:**
```bash
./start.sh
```

**或手动启动:**
```bash
# 终端1 - 后端
cd backend
npm start

# 终端2 - 前端
npm run dev
```

## 🆘 故障排查

### 无法访问管理后台

1. 检查前端服务是否运行
2. 清除浏览器缓存
3. 查看浏览器控制台错误

### 登录失败

1. 确认用户名密码正确
2. 检查后端服务是否运行
3. 查看后端日志
4. 检查 `backend/data/users.json`

### API错误

1. 检查后端服务状态
2. 运行测试脚本: `./test-api.sh`
3. 查看后端日志
4. 检查数据文件权限

### 端口冲突

如果端口被占用：

**前端 (5173):**
```bash
# 修改端口
PORT=5174 npm run dev
```

**后端 (3001):**
```bash
# 修改 backend/.env
PORT=3002
```
