## 添加音效文件

### 步骤 1: 下载音效
从上述网站下载 1-3 个音效文件（MP3 或 WAV 格式）

### 步骤 2: 放入项目目录
将下载的音效文件放入：
```
public/audio/boot/
```

### 步骤 3: 推荐文件名
```
public/audio/boot/
├── big-bang.mp3      # 主爆炸音效（推荐 2-5 秒）
├── cosmic-boom.wav   # 宇宙轰鸣
└── space-whoosh.mp3  # 太空呼啸
```

## 在代码中添加

### 修改 ClayBootAnimation.tsx

在组件开头添加音效播放：

```tsx
// 在 ClayBootAnimation 组件中添加
const boomSoundRef = useRef<HTMLAudioElement | null>(null);

useEffect(() => {
  // 初始化音效
  boomSoundRef.current = new Audio('/audio/boot/big-bang.mp3');
  boomSoundRef.current.volume = 0.8;

  return () => {
    boomSoundRef.current?.pause();
  };
}, []);

// 在爆炸阶段播放音效
if (frame === 60) {
  setPhase('explode');
  setExplodeFlash(1);
  // 播放爆炸音效
  boomSoundRef.current?.play().catch(console.error);
  particles.forEach((p) => {
    p.phase = 'exploding';
    p.vx = p.explosionVel.vx * 1.5;
    p.vy = p.explosionVel.vy * 1.5;
  });
}
```

## 音效时长建议

| 阶段 | 建议时长 | 音效类型 |
|------|---------|---------|
| 粒子聚集 | 0-2 秒 | 低频嗡鸣/渐强音 |
| 大爆炸 | 2-4 秒 | 爆炸轰鸣/冲击波 |
| Logo 显现 | 4-6 秒 | 空灵回响/叮铃声 |

## 浏览器自动播放策略

⚠️ **注意**: 浏览器可能阻止自动播放音频

**解决方案**:
1. 在用户与页面交互后（点击、按键）播放
2. 或者添加静音按钮让用户主动开启
