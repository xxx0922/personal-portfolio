# 个人作品集网站开发文档

## 第2部分：云服务器选择和购买

---

## 为什么需要云服务器

开发个人网站需要一台24小时运行的服务器来托管网站。云服务器相比传统服务器有以下优势：

1. **按需付费**：根据使用时间和配置付费，不需要一次性购买硬件
2. **快速部署**：几分钟内就能创建一台服务器
3. **稳定可靠**：云服务商提供高可用性保障
4. **弹性扩展**：随时可以升级配置
5. **全球部署**：可以选择离用户最近的机房

## 云服务商对比

### Google Cloud Platform（GCP）

**优点：**
- 全球网络基础设施优秀，国际访问速度快
- 新用户有300美元免费试用额度，有效期3个月
- 技术文档完善，社区活跃
- 服务稳定性好

**缺点：**
- 需要外币信用卡进行验证
- 界面和文档主要是英文
- 价格相对国内云服务商略高

**适合人群：**
- 网站面向国际用户
- 有外币信用卡
- 英文阅读能力较好

**推荐指数：** ⭐⭐⭐⭐⭐

### 阿里云

**优点：**
- 国内访问速度最快
- 中文界面和客服支持
- 价格实惠，新用户优惠力度大
- 与其他阿里云服务集成方便

**缺点：**
- 国际访问速度一般
- 使用国内机房需要域名备案（大约需要7-15天）
- 续费价格比首年价格高

**适合人群：**
- 网站主要面向国内用户
- 愿意完成域名备案流程
- 希望使用中文服务

**推荐指数：** ⭐⭐⭐⭐

### 腾讯云

**优点：**
- 国内访问速度快
- 新用户优惠活动多
- 与微信生态系统集成好
- 中文支持完善

**缺点：**
- 国内机房需要域名备案
- 国际访问速度一般
- 部分功能需要实名认证

**适合人群：**
- 需要与微信小程序等腾讯服务集成
- 主要面向国内用户
- 愿意完成备案

**推荐指数：** ⭐⭐⭐⭐

### AWS（亚马逊云）

**优点：**
- 全球最大的云服务商
- 服务种类丰富
- 新用户有12个月免费套餐

**缺点：**
- 学习曲线陡峭，初学者不太友好
- 价格较高
- 需要外币信用卡

**适合人群：**
- 有一定云计算经验
- 需要使用丰富的云服务
- 有外币信用卡

**推荐指数：** ⭐⭐⭐

## 服务器配置选择

### 最低配置（适合测试）

- **CPU**：1核
- **内存**：2GB
- **硬盘**：20GB SSD
- **带宽**：1Mbps
- **月费用**：约50-100元人民币
- **适用场景**：个人学习测试，访问量很小

### 推荐配置（本项目使用）

- **CPU**：2核
- **内存**：4GB
- **硬盘**：40GB SSD
- **带宽**：3-5Mbps
- **月费用**：约100-200元人民币
- **适用场景**：个人网站，日访问量1000以内

### 企业级配置

- **CPU**：4核或更多
- **内存**：8GB或更多
- **硬盘**：80GB SSD或更多
- **带宽**：10Mbps或更多
- **月费用**：约300元以上
- **适用场景**：商业网站，高并发访问

## Google Cloud购买流程

### 第一步：注册账号

1. 访问Google Cloud官网
2. 点击右上角的"开始免费试用"按钮
3. 使用Google账号登录（如果没有需要先注册Google账号）
4. 填写账单信息和信用卡信息进行验证
5. 成功后获得300美元免费额度，有效期3个月

**注意事项：**
- 需要外币信用卡（Visa或MasterCard）
- 信用卡会扣除1美元用于验证，之后会退还
- 免费额度用完或到期后才会开始计费

### 第二步：创建项目

1. 登录Google Cloud控制台
2. 点击顶部的项目选择器
3. 点击"新建项目"
4. 输入项目名称，比如"personal-portfolio"
5. 点击"创建"按钮

**说明：** Google Cloud使用项目来组织资源，一个账号可以创建多个项目。

### 第三步：创建虚拟机实例

1. 在左侧菜单找到"Compute Engine"（计算引擎）
2. 点击"VM实例"（虚拟机实例）
3. 点击"创建实例"按钮

**配置虚拟机：**

**基本信息：**
- 实例名称：输入一个好记的名称，比如"portfolio-server"
- 区域：选择离你的目标用户最近的区域
  - 如果面向中国用户，选择asia-east1（台湾）或asia-northeast1（日本）
  - 如果面向全球用户，选择us-central1（美国中部）

**机器配置：**
- 系列：选择E2（性价比高的通用型）
- 机器类型：选择e2-medium（2个vCPU，4GB内存）

**启动磁盘：**
- 点击"更改"按钮
- 操作系统：选择Ubuntu
- 版本：选择Ubuntu 20.04 LTS
- 磁盘类型：标准永久性磁盘
- 大小：40GB

**防火墙：**
- 勾选"允许HTTP流量"
- 勾选"允许HTTPS流量"

完成配置后点击"创建"按钮，等待几分钟虚拟机就会创建完成。

### 第四步：配置静态IP

默认情况下，虚拟机的IP地址是临时的，重启后会改变。我们需要将其改为静态IP：

1. 在左侧菜单找到"VPC网络"
2. 点击"外部IP地址"
3. 找到刚创建的虚拟机对应的IP地址
4. 类型那一列会显示"临时"，点击它
5. 选择"静态"
6. 输入一个名称，比如"portfolio-static-ip"
7. 点击"预留"按钮

**注意：** 静态IP会产生少量额外费用（约每月5-10美元），但对于网站来说是必需的，因为域名需要指向一个固定的IP。

### 第五步：记录服务器信息

创建完成后，记录以下信息：

- 外部IP地址：这是访问服务器的公网IP
- 实例名称：管理时需要用到
- 区域：了解服务器所在位置

## 阿里云购买流程

### 第一步：注册账号

1. 访问阿里云官网
2. 点击"免费注册"
3. 使用手机号完成注册
4. 登录后完成实名认证（上传身份证照片）

**注意：** 实名认证是必需的，通常1-2小时内审核完成。

### 第二步：购买ECS实例

1. 在阿里云官网搜索"云服务器ECS"
2. 点击"立即购买"
3. 进入配置页面

**购买配置：**

**计费方式：**
- 包年包月：适合长期使用，价格优惠
- 按量付费：按小时计费，灵活但总价更高
- 建议选择包年包月，购买1年

**地域和可用区：**
- 如果面向全国用户，选择华东、华北或华南
- 如果主要是本地用户，选择离你最近的地域
- 可用区选择随机分配即可

**实例规格：**
- 实例类型：选择"共享标准型"或"计算型"
- 规格：2核4GB（够用且性价比高）

**镜像：**
- 选择"公共镜像"
- 操作系统：Ubuntu
- 版本：Ubuntu 20.04 64位

**存储：**
- 系统盘：高效云盘，40GB

**网络：**
- 分配公网IP：勾选
- 带宽计费模式：按固定带宽
- 带宽：3Mbps或5Mbps

**安全组：**
- 使用默认安全组，或者新建安全组
- 确保开放以下端口：
  - 22端口（SSH远程连接）
  - 80端口（HTTP访问）
  - 443端口（HTTPS访问）
  - 3001端口（后端API，可选）

### 第三步：设置密码

- 设置实例名称：比如"portfolio-server"
- 设置root密码：务必使用强密码并妥善保存
- 主机名：可以使用默认值

### 第四步：确认订单

1. 勾选同意服务协议
2. 确认配置信息
3. 选择购买时长（建议1年）
4. 查看优惠券，新用户通常有折扣
5. 确认支付金额
6. 使用支付宝或微信支付

### 第五步：查看实例信息

支付成功后：

1. 进入阿里云控制台
2. 找到"云服务器ECS"
3. 点击"实例列表"
4. 可以看到刚购买的服务器

**记录以下信息：**
- 公网IP地址
- 实例ID
- root密码
- 区域信息

## 腾讯云购买流程

腾讯云的购买流程与阿里云非常相似：

### 第一步：注册和实名认证

1. 访问腾讯云官网
2. 注册账号（手机号或微信）
3. 完成实名认证

### 第二步：选择云服务器CVM

1. 进入云服务器CVM产品页
2. 点击"立即选购"
3. 选择配置（与阿里云类似）

**推荐配置：**
- 地域：根据目标用户选择
- 机型：标准型S5，2核4GB
- 镜像：Ubuntu Server 20.04
- 系统盘：50GB云硬盘
- 公网带宽：按流量计费或固定带宽3Mbps

### 第三步：设置安全组

确保安全组规则包含：
- 放通22端口（SSH）
- 放通80端口（HTTP）
- 放通443端口（HTTPS）

### 第四步：完成购买

设置实例密码，确认订单并支付。

## 服务器购买后的检查

无论选择哪个云服务商，购买完成后需要检查：

### 1. 确认服务器状态

在控制台查看实例状态是否为"运行中"。

### 2. 记录关键信息

- 公网IP地址（最重要）
- SSH登录用户名（通常是root或ubuntu）
- SSH登录密码
- 服务器所在区域

### 3. 测试网络连通性

在本地电脑打开命令行工具，尝试ping服务器IP：

在Windows：打开命令提示符
在Mac/Linux：打开终端

输入命令测试（将IP替换为你的服务器IP）：
ping 你的服务器IP

如果能收到回复，说明网络连通正常。

### 4. 检查防火墙规则

确认云服务商的安全组或防火墙已经开放了必要的端口。

## 常见问题

### 问题1：无法连接到服务器

**可能原因：**
- 安全组或防火墙没有开放22端口
- 输入的IP地址不正确
- 本地网络问题

**解决方法：**
- 检查云服务商控制台的安全组配置
- 确认IP地址是否正确
- 尝试使用手机热点连接测试

### 问题2：忘记root密码

**解决方法：**
- 在云服务商控制台可以重置密码
- 阿里云：实例列表 → 更多 → 密码/密钥 → 重置密码
- GCP：需要使用SSH密钥登录，可以在控制台生成密钥

### 问题3：如何选择服务器地域

**建议：**
- 如果用户主要在中国，选择国内地域（但需要备案）
- 如果不想备案，可以选择香港或海外地域
- 如果追求访问速度，选择离用户最近的地域

### 问题4：配置不够用怎么办

**解决方法：**
- 大多数云服务商支持在线升级配置
- 阿里云和腾讯云可以直接升级
- GCP可以停止实例后更改机器类型

## 成本估算

### 国内云服务商（阿里云/腾讯云）

**最低配置：**
- 1核2GB，1Mbps带宽
- 价格：约60-80元/月

**推荐配置：**
- 2核4GB，3-5Mbps带宽
- 价格：约120-180元/月
- 新用户首年通常有5-7折优惠

### Google Cloud Platform

**推荐配置：**
- e2-medium（2核4GB）
- 40GB磁盘
- 价格：约30-40美元/月（约210-280元人民币）
- 新用户有300美元免费额度，可以用6-10个月

### 节省成本的技巧

1. **利用新用户优惠**：首次购买通常有大幅折扣
2. **包年购买**：比按月购买便宜20-30%
3. **关注促销活动**：双十一、618等活动期间折扣力度大
4. **合理选择带宽**：带宽是费用大头，按需选择即可
