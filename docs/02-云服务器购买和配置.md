## 云服务器选择

### 主流云服务商对比

#### 1. Google Cloud Platform (GCP)
- **优点**：
  - 全球网络性能优秀
  - 免费试用额度（$300，3个月）
  - 国际访问速度快
  - 技术文档完善
- **缺点**：
  - 需要外币信用卡
  - 界面为英文
  - 价格相对较高
- **推荐指数**：⭐⭐⭐⭐⭐

#### 2. 阿里云
- **优点**：
  - 国内访问速度快
  - 中文界面和客服
  - 价格实惠
  - 新用户优惠力度大
- **缺点**：
  - 国际访问速度一般
  - 需要域名备案（国内）
- **推荐指数**：⭐⭐⭐⭐

#### 3. 百度智能云
- **优点**：
  - 国内访问速度快
  - 新用户优惠
  - 与微信生态集成好
- **缺点**：
  - 需要域名备案（国内）
  - 国际访问速度一般
- **推荐指数**：⭐⭐⭐⭐

#### 4. AWS（Amazon Web Services）
- **优点**：
  - 全球最大的云服务商
  - 服务种类丰富
  - 免费套餐（12个月）
- **缺点**：
  - 价格较高
  - 学习曲线陡峭
  - 需要外币信用卡
- **推荐指数**：⭐⭐⭐⭐

### 服务器配置推荐

#### 个人网站最低配置
- **CPU**：1 核
- **内存**：2GB RAM
- **硬盘**：20GB SSD
- **带宽**：1Mbps
- **月费用**：约 ¥50-100

#### 推荐配置（本项目使用）
- **CPU**：2 核
- **内存**：4GB RAM
- **硬盘**：40GB SSD
- **带宽**：3-5Mbps
- **月费用**：约 ¥100-200

#### 企业级配置
- **CPU**：4 核或更多
- **内存**：8GB RAM 或更多
- **硬盘**：80GB SSD 或更多
- **带宽**：10Mbps 或更多
- **月费用**：约 ¥300+

### 百度智能云购买流程

#### 步骤 1：注册百度智能云账号

1. 访问 (https://login.bce.baidu.com/)
2. 点击右上角 **"免费注册"**
3. 使用手机号注册
4. 完成实名认证

#### 步骤 2：购买 ECS 实例

1. 进入 **"云服务器 ECS"** 产品页
2. 点击 **"立即购买"**
3. 选择配置：
   - **计费方式**：按量付费或包年包月
   - **地域和可用区**：选择离用户最近的地域
   - **实例规格**：
     - 2 vCPU
     - 4 GiB 内存
   - **镜像**：公共镜像 → Ubuntu 20.04 64位
   - **存储**：40GB 高效云盘
   - **网络**：
     - 分配公网 IP
     - 带宽：按固定带宽（3-5Mbps）
   - **安全组**：
     - 放行 22（SSH）
     - 放行 80（HTTP）
     - 放行 443（HTTPS）
     - 放行 3001（后端 API，可选）
4. 设置实例名称和密码
5. 确认订单并支付

#### 步骤 3：查看实例信息

1. 进入 **"控制台"** → **"云服务器 ECS"**
2. 查看实例列表
3. 记录公网 IP 地址

## 连接到服务器

### Windows 用户

#### 方法 1：使用 PuTTY

1. **下载安装 PuTTY**
   - 访问 https://www.putty.org/
   - 下载并安装

2. **连接服务器**
   - 打开 PuTTY
   - Host Name：输入服务器公网 IP
   - Port：22
   - Connection type：SSH
   - 点击 **"Open"**
   - 首次连接会弹出安全警告，点击 **"是"**
   - 输入用户名：`root`
   - 输入密码（输入时不显示）

#### 方法 2：使用 Windows Terminal + SSH

1. **安装 Windows Terminal**（Windows 10/11）
   - 从 Microsoft Store 安装

2. **连接服务器**
   ```bash
   ssh root@你的服务器IP
   ```
   - 首次连接输入 `yes` 确认
   - 输入密码


### 配置 SSH 密钥登录（推荐）

#### 1. 生成 SSH 密钥对（本地）

```bash
# 生成密钥
ssh-keygen -t rsa -b 4096 -C "your_email@example.com"

# 按提示操作：
# - 保存位置：直接回车使用默认位置 (~/.ssh/id_rsa)
# - 密码：可以设置也可以留空
```

#### 2. 上传公钥到服务器

```bash
# 方法 1：使用 ssh-copy-id（推荐）
ssh-copy-id root@你的服务器IP

# 方法 2：手动复制
# 本地查看公钥
cat ~/.ssh/id_rsa.pub

# 登录服务器后执行
mkdir -p ~/.ssh
echo "你的公钥内容" >> ~/.ssh/authorized_keys
chmod 600 ~/.ssh/authorized_keys
chmod 700 ~/.ssh
```

#### 3. 测试密钥登录

```bash
ssh root@你的服务器IP
# 应该不需要输入密码直接登录
```

## 服务器基础配置

### 1. 更新系统

```bash
# 更新软件包列表
sudo apt update

# 升级已安装的软件包
sudo apt upgrade -y

# 清理不需要的包
sudo apt autoremove -y
```

### 2. 设置时区

```bash
# 查看当前时区
timedatectl

# 设置为中国时区
sudo timedatectl set-timezone Asia/Shanghai

# 验证
date
```

### 3. 创建非 root 用户（可选但推荐）

```bash
# 创建新用户
sudo adduser deploy

# 添加到 sudo 组
sudo usermod -aG sudo deploy

# 切换到新用户
su - deploy

# 为新用户配置 SSH 密钥
mkdir -p ~/.ssh
sudo cp /root/.ssh/authorized_keys ~/.ssh/
sudo chown deploy:deploy ~/.ssh/authorized_keys
chmod 600 ~/.ssh/authorized_keys
```

### 4. 配置主机名

```bash
# 设置主机名
sudo hostnamectl set-hostname portfolio-server

# 编辑 hosts 文件
sudo nano /etc/hosts

# 添加以下内容：
# 127.0.0.1 portfolio-server
```

## 安装必需软件

### 1. 安装 Node.js

#### 使用 NodeSource 仓库

```bash
# 下载并执行 NodeSource 安装脚本（安装 Node.js 20.x）
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -

# 安装 Node.js
sudo apt install -y nodejs

# 验证安装
node --version  # 应该显示 v20.x.x
npm --version   # 应该显示 10.x.x
```

### 2. 安装 Nginx

```bash
# 安装 Nginx
sudo apt install -y nginx

# 启动 Nginx
sudo systemctl start nginx

# 设置开机自启
sudo systemctl enable nginx

# 查看状态
sudo systemctl status nginx

# 验证安装
nginx -v  # 应该显示版本号
```

访问服务器 IP，应该能看到 Nginx 欢迎页面。

### 3. 安装 PM2

```bash
# 全局安装 PM2
sudo npm install -g pm2

# 验证安装
pm2 --version

# 设置 PM2 开机自启
pm2 startup
# 按照提示执行命令
```

### 4. 安装 Git

```bash
# 安装 Git
sudo apt install -y git

# 验证安装
git --version

# 配置 Git（可选）
git config --global user.name "Your Name"
git config --global user.email "your_email@example.com"
```

### 5. 安装其他工具

```bash
# 安装常用工具
sudo apt install -y \
    curl \
    wget \
    vim \
    nano \
    htop \
    unzip \
    build-essential

# 验证
curl --version
wget --version
vim --version
```

## 配置防火墙

### 使用 UFW（Ubuntu Firewall）

#### 1. 安装和启用 UFW

```bash
# 安装 UFW（Ubuntu 默认已安装）
sudo apt install -y ufw

# 设置默认规则
sudo ufw default deny incoming   # 拒绝所有入站
sudo ufw default allow outgoing   # 允许所有出站

# ⚠️ 重要：先允许 SSH，否则会断开连接
sudo ufw allow 22/tcp
# 或者更安全的方式：
sudo ufw allow from any to any port 22 proto tcp

# 启用防火墙
sudo ufw enable

# 查看状态
sudo ufw status verbose
```

#### 2. 开放必需端口

```bash
# HTTP
sudo ufw allow 80/tcp

# HTTPS
sudo ufw allow 443/tcp

# 后端 API（可选，如果需要直接访问）
sudo ufw allow 3001/tcp

# 查看规则
sudo ufw status numbered
```

#### 3. 删除规则（如果需要）

```bash
# 查看规则编号
sudo ufw status numbered

# 删除指定编号的规则
sudo ufw delete [编号]

# 例如：删除规则 3
sudo ufw delete 3
```

### 云服务商安全组配置

如果使用云服务器，还需要在云服务商控制台配置安全组：

#### Google Cloud Platform

1. 进入 **"VPC 网络"** → **"防火墙规则"**
2. 创建防火墙规则：
   - **名称**：`allow-http-https`
   - **目标**：网络中的所有实例
   - **源 IP 范围**：`0.0.0.0/0`
   - **协议和端口**：
     - tcp: 80
     - tcp: 443
3. 点击 **"创建"**

#### 百度智能云

1. 进入 **"云服务器 ECS"** → **"安全组"**
2. 选择实例的安全组
3. 点击 **"配置规则"** → **"入方向"**
4. 添加规则：
   - **端口范围**：80/80
   - **授权对象**：0.0.0.0/0
   - **描述**：HTTP
5. 添加规则：
   - **端口范围**：443/443
   - **授权对象**：0.0.0.0/0
   - **描述**：HTTPS

## 服务器安全加固（可选）

### 1. 禁用 root 密码登录

```bash
# 编辑 SSH 配置
sudo nano /etc/ssh/sshd_config

# 修改以下配置：
# PermitRootLogin yes → PermitRootLogin prohibit-password
# PasswordAuthentication yes → PasswordAuthentication no

# 重启 SSH 服务
sudo systemctl restart sshd
```

### 2. 安装 Fail2Ban（防暴力破解）

```bash
# 安装 Fail2Ban
sudo apt install -y fail2ban

# 启动服务
sudo systemctl start fail2ban
sudo systemctl enable fail2ban

# 查看状态
sudo fail2ban-client status
```

### 3. 配置自动安全更新

```bash
# 安装 unattended-upgrades
sudo apt install -y unattended-upgrades

# 启用自动更新
sudo dpkg-reconfigure -plow unattended-upgrades
```
