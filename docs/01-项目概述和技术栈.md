## 项目简介

这是一个全栈个人作品集网站，用于展示个人项目经验、技能、博客文章、照片等内容。网站包含前台展示和后台管理系统，支持动态内容管理。

### 项目信息

- **项目名称**：Personal Portfolio（个人作品集）
- **部署域名**：https://www.bohenan.com
- **项目类型**：全栈 Web 应用
- **开发者**：个人项目

## 功能特性

### 前台功能

#### 1. 首页展示
- ✅ 个人信息展示（头像、姓名、职位、联系方式）
- ✅ 照片墙自动轮播（6秒切换）
- ✅ 响应式设计，支持移动端

#### 2. 内容模块
- ✅ **博客文章**：展示技术文章，支持分类和标签
- ✅ **新闻动态**：展示个人动态和成就
- ✅ **项目经验**：展示项目详情、技术栈、合同金额、项目图片和附件
- ✅ **技能展示**：按分类展示技能树，带熟练度评级
- ✅ **影音书籍**：展示看过的电影和书籍，带评分和评论
- ✅ **精彩瞬间**：瀑布流照片墙，支持灯箱预览
- ✅ **知识文档**：技术文档展示，支持附件下载
- ✅ **工作经历**：时间轴展示工作经历

#### 3. 特色功能
- ✅ **背景音乐播放器**：支持多首歌曲循环播放，上一首/下一首切换
- ✅ **联系表单**：在线留言功能
- ✅ **友情链接**：底部友情链接展示
- ✅ **图片懒加载**：优化页面加载性能
- ✅ **SEO 优化**：自定义 meta 标签，支持社交媒体分享

### 后台管理功能

#### 1. 内容管理
- ✅ **项目管理**：添加、编辑、删除项目，上传项目图片和附件，支持私密项目
- ✅ **技能管理**：按分类管理技能，设置熟练度
- ✅ **博客文章管理**：Markdown 编辑，分类标签管理
- ✅ **新闻动态管理**：发布新闻，设置置顶
- ✅ **照片管理**：单张上传和批量上传，分类管理
- ✅ **文档管理**：上传文档附件（最大50MB）
- ✅ **影音书籍管理**：添加电影/书籍评价，上传封面和附件

#### 2. 系统配置
- ✅ **导航菜单管理**：自定义导航菜单
- ✅ **友情链接管理**：添加友情链接，上传logo
- ✅ **Footer 设置**：自定义底部内容
- ✅ **社交媒体管理**：配置社交媒体链接
- ✅ **标签管理**：统一管理所有标签
- ✅ **背景音乐设置**：上传音乐，创建播放列表
- ✅ **个人信息管理**：编辑个人资料，上传照片墙

#### 3. 数据统计
- ✅ **访问统计**：查看页面访问量
- ✅ **数据概览**：各类内容数量统计
- ✅ **留言管理**：查看和管理用户留言

#### 4. 文件管理
- ✅ **文件上传**：支持图片、音频、PDF、Word、Excel 等多种格式
- ✅ **文件列表**：查看所有上传的文件
- ✅ **图片压缩**：自动生成缩略图
- ✅ **大文件支持**：最大支持 50MB 文件上传

## 技术栈

### 前端技术栈

#### 核心框架
- **React 18**：前端 UI 框架
- **TypeScript**：类型安全的 JavaScript 超集
- **Vite**：快速的前端构建工具

#### 路由和状态管理
- **React Router v6**：客户端路由
- **React Hooks**：状态管理（useState, useEffect, useRef 等）

#### UI 组件库
- **Tailwind CSS**：实用优先的 CSS 框架
- **React Masonry CSS**：瀑布流布局
- **LazyImage**：自定义图片懒加载组件

#### 工具库
- **Axios**：HTTP 客户端（用于 API 请求）
- **React Markdown**：Markdown 渲染
- **date-fns**：日期处理

### 后端技术栈

#### 核心框架
- **Node.js 18+**：JavaScript 运行时
- **Express.js**：Web 应用框架

#### 数据存储
- **JSON 文件存储**：简单的文件系统数据库
- **文件系统 (fs/promises)**：异步文件操作

#### 认证和安全
- **JWT (jsonwebtoken)**：用户认证
- **bcryptjs**：密码加密
- **cors**：跨域资源共享配置

#### 文件处理
- **Multer**：文件上传中间件
- **Sharp**：图片处理和压缩库

#### 其他工具
- **body-parser**：请求体解析
- **dotenv**：环境变量管理

### 部署技术栈

#### 服务器环境
- **操作系统**：Ubuntu 20.04 LTS
- **Web 服务器**：Nginx
- **进程管理**：PM2
- **反向代理**：Nginx proxy_pass

#### 域名和 SSL
- **域名注册商**：阿里云/腾讯云等
- **DNS 解析**：云服务商 DNS
- **SSL 证书**：Let's Encrypt（免费）
- **HTTPS 配置**：Nginx SSL 配置

#### 云服务器
- **服务商**：Google Cloud / 阿里云 / 腾讯云
- **实例类型**：2 核 CPU，4GB 内存，40GB 硬盘
- **网络**：固定公网 IP
- **防火墙**：开放 80（HTTP）和 443（HTTPS）端口

## 开发环境要求

### 本地开发

#### 必需软件
1. **Node.js**：v20.19+ 或 v22.12+（推荐使用 nvm 管理版本）
2. **npm**：v10+ 或 **yarn**：v1.22+
3. **Git**：v2.30+
4. **代码编辑器**：VS Code（推荐）或其他

#### VS Code 推荐插件
- ESLint：代码检查
- Prettier：代码格式化
- Tailwind CSS IntelliSense：Tailwind 自动补全
- TypeScript Vue Plugin：TypeScript 支持
- GitLens：Git 增强

#### 推荐的系统配置
- **操作系统**：Windows 10/11、macOS 或 Linux
- **内存**：至少 8GB RAM
- **硬盘**：至少 10GB 可用空间

### 生产环境

#### 服务器要求
- **操作系统**：Ubuntu 20.04 LTS 或更高版本
- **CPU**：2 核或更多
- **内存**：4GB RAM 或更多
- **硬盘**：40GB SSD 或更多
- **网络**：固定公网 IP 地址

#### 必需软件（服务器）
1. **Node.js**：v20.19+ 或 v22.12+
2. **Nginx**：v1.18+
3. **PM2**：进程管理工具
4. **Git**：用于代码部署
5. **Certbot**：Let's Encrypt SSL 证书工具

#### 域名要求
- 已注册的域名
- DNS 解析已配置到服务器 IP
- 支持 A 记录和 CNAME 记录
